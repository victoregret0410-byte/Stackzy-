Politique de Confidentialité — Stackzy

Dernière mise à jour : avril 2026

Stackzy ("nous", "notre", "l'application") respecte votre vie privée. Cette politique explique quelles données sont collectées et comment elles sont utilisées.

1. Données collectées

Données de jeu : votre score, progression, skins débloqués et préférences de jeu sont stockés localement sur votre appareil. Si vous créez un compte, ces données sont synchronisées sur nos serveurs pour permettre la sauvegarde cloud.

Identifiant : si vous créez un compte, nous stockons votre adresse email et votre pseudo. Ces données sont utilisées uniquement pour l'authentification et l'affichage dans les classements.

Achats : les achats in-app sont traités par Apple via le système StoreKit. Nous ne stockons aucune information de carte bancaire.

2. Publicités

L'application affiche des publicités via Google AdMob. AdMob peut collecter des identifiants publicitaires (IDFA) pour afficher des publicités pertinentes. Vous pouvez désactiver le suivi publicitaire dans les réglages de votre appareil. Vous pouvez supprimer toutes les publicités via un achat in-app.

3. Partage des données

Nous ne vendons, ne louons et ne partageons aucune donnée personnelle avec des tiers, sauf :

- Google AdMob : pour l'affichage de publicités (peut être désactivé par achat)
- Apple : pour le traitement des achats in-app
- Supabase : pour la sauvegarde cloud (si vous créez un compte)

4. Droits de l'utilisateur

Vous pouvez à tout moment :

- Supprimer votre compte et toutes vos données en nous contactant
- Réinitialiser vos données locales dans les paramètres de l'application
- Refuser le suivi publicitaire via les réglages iOS

5. Données des enfants

L'application est classée 4+ et ne collecte pas sciemment de données personnelles d'enfants de moins de 13 ans au-delà de ce qui est nécessaire au fonctionnement du jeu.

6. Sécurité

Les données sont transmises via HTTPS et stockées sur des serveurs sécurisés. L'authentification utilise des liens magiques (magic links) sans stockage de mots de passe.

7. Contact

Pour toute question concernant cette politique :
Email : privacy@stackmaster.app

8. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut du document.
